# VpK Snapshot

A desktop app built on Electron for creating a snapshot of a kubernetes cluster.  The output is then read
by VpK (Visual parsed Kubernetes).

## Usage

### Install Dependencies

```
npm install
```

### Run

```
npm start
npm run dev (with Nodemon)
```

### Package

```
npm run package-mac
npm run package-win
npm run package-linux
```
